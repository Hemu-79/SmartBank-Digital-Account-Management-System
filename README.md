# SmartBank-Digital-Account-Management-System

SmartBank is a desktop-based banking management application developed in Java using Swing. The project simulates core banking operations such as account creation, deposits, withdrawals, and account management through an interactive graphical user interface. The system is designed using Object-Oriented Programming principles and demonstrates practical implementation of file handling, exception handling, and modular software design.

Key Features

✅ Multiple Account Types

Savings Account
Student Account
Current Account

✅ Banking Operations

Create new accounts
Deposit funds
Withdraw funds
View account details
Manage customer records

✅ Data Persistence

Stores account information using binary file storage
Maintains data across application sessions

✅ Error Handling

Prevents invalid transactions
Handles insufficient balance scenarios
Validates user inputs and account information

Application Workflow
User launches the application.
Selects the desired banking operation.
Enters account details through GUI forms.
System validates inputs and processes requests.
Data is stored and retrieved using file handling mechanisms.
Results are displayed through the Swing interface.
Project Highlights
Designed using modular class architecture.
Implemented Object-Oriented concepts including Inheritance, Encapsulation, and Polymorphism.
Developed an intuitive GUI using Java Swing.
Applied file-based persistence to maintain account records.
Improved reliability through exception handling and input validation.
